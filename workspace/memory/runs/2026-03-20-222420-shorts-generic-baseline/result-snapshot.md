# Result Snapshot

## New Constraint
- Shorts 固定預設必須是通用型，不綁 `POKOPIA` 或任何單一題材

## What Changed
- 把 `voice_memory` 裡偏個案的 `POKOPIA 豪窄稿` 規則改寫成通用原則
- 明確記下：之後要學的是句長、帶路感、結果感、收尾方式
- 明確禁止把舊稿的專有名詞、場景或梗直接搬去新題目

## Generic Principles
- 第一拳先丟最具體、最好懂的畫面或結果
- `intro` 要短，快速進入帶路句
- 優先用 `小條件 -> 大結果` 的反差句型
- 能具體就具體，盡量落成數字、條件、次數、結果
- 正文維持第一人稱帶路感
- 最後用 `成果重講一次 + 很短的推進` 收尾
