# Result Snapshot

## Topic Summary
- 使用者指定的題目 `自動瀑布門 / 水簾洞感應門` 有足夠機制證據可寫
- 這題不是純幻想造景，社群已實作出：
  - `laser sensor`
  - `door / window chain`
  - `floodgate / window hatch`
  - `automatic lava / waterfall gate`

## Platform Outcome
- `Reddit / 英文媒體 / 英文 guide`：訊號最強
- `中文站外`：有 `搬水 / 人造瀑布 / 無限水源`，但 `感應瀑布門` 精準樣本偏薄
- `YouTube Shorts / TikTok / IG Reels / 巴哈`：公開可索引的精準樣本不足，已查但資料不強

## Writing Direction
- 不要把這題寫成純技術解說
- 主 punch 要寫成：
  - `Pokopia 連瀑布都能做成感應自動門`
- 最該講的 3 個點：
  - 先解鎖 `搬水`
  - `感應器` 不是看按鈕，是看移動
  - 訊號要靠 `門 / 窗蓋` 接力傳到上方的 `水門`
