# Request

記錄存檔，之後產出 `Shorts` 文稿固定只交：

- `Hook Title`
- `Hook Burst Text`
- `Template Marked Script`

並沿用這些規則：

- `Template Marked Script` 用 `[hook] / [intro] / [chapter] / [ending]`
- `[chapter]` 寫成可直接念的 `第一步`、`第二步` 這種格式
- 後製重點字用 `「」`
- 只標 `intro` 和章節正文
- 不標 `hook`
- 不標 `[chapter]` 那一行本身
- 只標少量真正重要的名詞
