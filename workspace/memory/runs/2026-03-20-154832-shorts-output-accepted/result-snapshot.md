# Result Snapshot

- 已把使用者最新確認的 `Shorts` 文稿格式正式存檔。
- 之後 `Shorts` 正式交付固定只交：
  - `Hook Title`
  - `Hook Burst Text`
  - `Template Marked Script`
- `Template Marked Script` 固定直接標：
  - `[hook]`
  - `[intro]`
  - `[chapter] 第一步 ...`
  - `[ending]`
- `[chapter]` 後面不能寫 `第一章`，要改成自然可念的步驟或重點。
- 後製重點字固定用 `「」`。
- 重點字只標 `intro` 和章節正文，不標 `hook`，也不標 `[chapter]` 那一行本身。
- 一次只標少量關鍵名詞，不標整句。
- 舊的多欄位格式：
  - `Intro Overlay Text`
  - `Step Cards`
  - `Teaching Subtitle Lines`
  - `Final Short Script`
  已降級成歷史版本，不再當正式輸出格式。
