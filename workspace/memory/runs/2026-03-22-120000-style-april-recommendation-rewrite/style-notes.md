# Style Notes

這次使用者親改版和先前通用攻略稿相比，新增出來的可重用方向：

1. 這不是純攻略片，而是 `新作推薦片`。
2. 新作推薦片不需要每段都用強烈的 `我現在直接教你怎麼做`。
3. 但仍然要像在幫觀眾快速挑片，而不是照著商店頁念簡介。
4. 每段最常見的有效結構是：
   - 先說這款最有記憶點的地方
   - 再補角色 / 世界觀 / 核心玩法
   - 最後一句幫觀眾下判斷
5. 可以比攻略稿更敢用：
   - `超可愛`
   - `超級震撼`
   - `最棒的舒壓與搞笑擔當`
   - `必玩的強作`
   但一定要有具體內容撐住。
6. `我們要扮演`、`我們會扮演` 在推薦片裡很自然，不會破壞聲線。
