# Pokemon Champions Game Memory

## Last Updated
- `2026-03-30`

## Core Positioning
- `Pokémon Champions` 是一款 `battle-focused` 的寶可夢對戰遊戲，不是正統抓怪推主線型 RPG。
- 主要受眾是 `VGC / 排位 / 組隊 / 單打雙打對戰玩家`。
- 如果之後要做 `買前必看 / 入坑判斷` 類型，核心一定要先回答：
  - 它到底是不是你的菜
  - 它的免費制和付費架構怎麼看
  - HOME 連動有哪些限制

- 官方玩法與商品頁可直接確認：
  - 對戰模式有 `Ranked / Casual / Private`

  - 對戰證與 `SBS` 每賽季重設
  - `VP` 是錄用與培育的核心資源，且官方英文頁明寫 `VP cannot be directly purchased`

## Monetization
- 本體免費，但更準確的理解是 `免費入場 + 後續付費架構完整`：
  - `起始包`
  - `特級對戰證`
  - `會員`
- 繁中官網價格：
  - 起始包 `200 TWD`
  - 特級對戰證 `320 TWD / 每賽季`
  - 會員 `160 TWD / 月`
  - 會員 `1,600 TWD / 年`
- 會員功能：
  - 擴充盒子容量
  - 增加對戰隊伍數量
  - 會員專屬任務
  - 追加戰鬥音樂

## Battle Structure
- 對戰模式：
  - `Ranked`
  - `Casual`
  - `Private`
- 格式：
  - `Single Battle`
  - `Double Battle`
- `VP` 是核心資源：
  - 用來錄用與訓練寶可夢
  - `不能直接購買`
- 對戰證與 `SBS` 會每賽季重設，這是一款明確的賽季制作品

## Pokemon Acquisition and Training
- 不是抓寶可夢，而是 `錄用`
- `Trial Recruitment` 的核心用途是先測隊，再決定要不要正式投入
- 隨機名單可 `22 小時` 免費刷新一次
- 培育可直接調整：
  - 能力
  - 特性
  - 招式
- `Trial Recruitment` 狀態下無法訓練

## HOME Limits
- 只有在《Pokémon Champions》登場的寶可夢可從 HOME 帶入
- Champions 內錄用的寶可夢不能回 HOME
- Champions 內神秘禮物等取得的寶可夢不能回 HOME
- Champions 內訓練結果不會直接反映到 HOME
- 同一隻訪客寶可夢如果換型態再來，先前在 Champions 的訓練內容會重置

## Competitive Framing
- 首套級別對戰規則開放 `超級進化`
- `超級快龍` 已確認會在本作登場
- `2026` 年起世界賽遊戲部門指定使用《Pokémon Champions》
- 賽季與規則會輪替，這是一款 `長線營運競技遊戲`

## Community / Preview Takeaways
- 海外社群目前最常用的比喻是：`官方版 Showdown / 官方 battle simulator`。
- 媒體試玩與 VGC 討論目前反覆提到：
  - `team preview` 的資訊整理感更重
  - 招募有稀有演出與稱號個體話題
  - 招式說明會直接標示更精確的數字與機率

  - `Protect 8PP` 這類 PP 下修正在被當成節奏重做訊號

## 2026-04-02 Prelaunch Topic Signal

- 發售前最適合台灣觀眾的長片角度之一，不是純教學，而是：
  - `老玩家優勢有多大？發售前新手最該先準備的幾件事`
- 這題能打中的原因：
  - 台灣觀眾最在意 `起跑線公平性`
  - 巴哈焦慮集中在 `課金 / VP / HOME / 老玩家優勢`
  - 中文圈、英文圈、日文圈都反覆出現 `事前準備 / beginner guide / 課金疑慮`
- 最佳切角不是：
  - `新手打不贏老手`
- 而是：
  - `老玩家確實有優勢，但《寶可夢冠軍》這次也有在縮短差距；新手真正該準備的是對的東西`

## 2026-04-16 Beginner-Friendly Simple Team Memory

- 使用者提供的日文高分逐字稿，可回用成一支很穩的 `新手也能玩的超簡單隊伍` 長片。
- 這套隊伍可整理成：
  - `Mega噴火龍Y`
  - `烈咬陸鯊`
  - `甲賀忍蛙`
  - `西獅海壬`
  - `Mega巨鉗螳螂`
  - `仆刀將軍`
- 最好記的片內主結論不是：
  - `哪一隻最強`
- 而是：
  - `這隊的分工非常清楚，新手可以先照 SOP 打`
- 最穩的預設選出：
  - `Mega噴火龍Y + 烈咬陸鯊 + 仆刀將軍`
- 對面若有明顯多隻撒岩點，最穩的替代選出：
  - `甲賀忍蛙 + Mega巨鉗螳螂 + 烈咬陸鯊`
- 這隊最適合的腳本 promise：
  - `你不用很會做複雜輪轉，也能先把單打節奏打清楚`
  - `這不是花式炫技隊，而是一套角色分工很直覺的入門模板`
- 最好用的正文框架：
  - `前排開路`
  - `中段鋪傷`
  - `後排收頭`
- 影片裡最該用白話講清楚的 4 個點：
  - `Mega噴火龍Y 負責開壓，不是每場都硬帶`
  - `烈咬陸鯊是中樞，不只是打手`
  - `仆刀將軍 / 巨鉗螳螂都是收尾位，但節奏不同`
  - `怕撒岩就改走忍蛙 + 螳螂線`

## 2026-04-16 Charizard Y Team Deep Integration Memory

- 如果之後再做 `Mega噴火龍Y` 隊伍片，不要只沿用單支影片。
- 目前最穩的做法是整合兩支 `れしむ` 高分影片：
  - `IKCPLvLCjP4`
  - `pJyJUQyHfeg`
- 這兩支片的最佳中文整理結論：
  - `這不是兩套完全不同的隊`
  - `而是一套核心三角固定、後三格持續修正的高分模板`
- 可回用的固定核心：
  - `Mega噴火龍Y`
  - `烈咬陸鯊`
  - `仆刀將軍`
- 可回用的支援位演進：
  - 早期：
    - `阿羅拉九尾`
    - `西獅海壬`
    - `Mega花葉蒂`
  - 作者留言補充：
    - `阿羅拉九尾` 有時改成 `氣勢披帶甲賀忍蛙`
    - `沒有 Mega花葉蒂` 可用 `Mega大竺葵`
  - 後期高分版：
    - `西獅海壬`
    - `甲賀忍蛙`
    - `Mega巨鉗螳螂`

  - `勝利流程`
  - `預設選出`
  - `什麼時候不要硬帶噴火龍`
  - `替代方案`
