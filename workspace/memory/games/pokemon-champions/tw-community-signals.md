# Taiwan Community Signals

## Last Updated

- `2026-04-09`

## Core Read

- 台灣社群現在最大的問題不是 `有沒有興趣`
- 而是 `想進，但怕被勸退`
- 真正的焦慮源頭集中在 4 件事：
  - `不課金行不行`
  - `沒有 HOME 行不行`
  - `第一天會不會亂花資源`
  - `純 PvP 平台到底適不適合一般寶可夢玩家`

## Most Wanted

- `沒有 HOME 到底能不能玩`
- `不課金到底能不能追得上`
- `第一天先做什麼`
- `手機版什麼時候上`
- `這款到底在玩什麼，跟本傳差在哪`
- `要不要先回本傳準備寶可夢`

## Biggest Concerns

- `起跑線公平性`
- `老玩家優勢`
- `課金壓力`
- `遊戲太硬、太競技、太小眾`
- `手機版沒同步`
- `台灣區購買與商城流程不直觀`

## Pain Points

- 中文資訊很多，但很碎
- 台灣新手缺的是 `第一天流程圖`，不是再一支泛介紹
- 一旦牽涉 `HOME / 通行證 / 起始包 / VP`，觀眾就很容易進入 `先觀望` 狀態
- 純 PvP 讓很多本傳玩家懷疑自己是不是目標受眾

## Best Script Framing

- 不要把它講成 `這遊戲有多酷`
- 要講成 `台灣新手最怕的幾件事，我先幫你拆掉`
- 最有效的主 promise：
  - `你不是不能入坑，你只是很容易被這幾個點嚇到`

## Good Chapter Order

1. `這款不是本傳，是純對戰平台`
2. `不課金 / HOME / 老玩家優勢一次講清楚`
3. `第一天最容易踩雷的地方`
4. `你到底是不是適合這款的人`
