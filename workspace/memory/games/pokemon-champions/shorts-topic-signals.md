## Strongest Repeatable Shorts Clusters

1. `HOME 根本不是你以為的傳送`
2. `起始包 / 票券 / 課金值不值`
3. `Shiny / 閃光也能玩`
4. `首日最扯 BUG / 不滿點`
5. `3 隻現在該認識的 Mega / 免費 Mega 石`

## Best Top 1 For Chinese Audience

- `HOME 真相`
- 理由：
  - 三語圈都有
  - 符合頻道的 `拆誤解 / 拆焦慮 / 實用型` 定位
  - 不需要高端對戰畫面也能成立

## Platform Takeaways

- `YouTube Shorts`
  - 直接搜遊戲名不夠，之後固定補 `site:youtube.com/shorts`
- `TikTok`
  - 本輪 `opencli tiktok search` 不穩，之後記得用 `site:tiktok.com` 補查
- `IG Reels`
  - 很適合找 `重要觀念 / 官方沒講清楚 / 新手誤解`
- `bilibili`
  - 適合找 `Mega / 強角 / 套路 / 模板隊`
- `巴哈`
  - 適合驗證中文社群是不是也在吵同一個點

## Topic Packaging Defaults

- 如果是 `HOME` 題：
  - 先打錯誤認知
  - 再用一句話翻轉
  - 最後補一個後果
- 如果是 `課金 / 票券` 題：
  - 用 `3 種券差在哪` 的格式最好做
- 如果是 `Mega` 題：
  - 用 `3 隻你該先認識`、`8 顆免費 Mega 石`
- 如果是 `Bug / 不滿點` 題：
  - 時效短，優先搶首週
