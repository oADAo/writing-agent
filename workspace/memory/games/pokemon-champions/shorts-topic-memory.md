# Pokemon Champions Shorts Topic Memory

## Last Updated

- `2026-03-31`

## Shorts Market Snapshot

- 以 `2026-03-31` 來看，《寶可夢冠軍》的中文原生 `YouTube Shorts` 樣本仍偏少。
- 目前最有訊號的短影音平台是：
  - `日文 TikTok`
  - `英文 TikTok`
  - `英文 IG Reels`
- 中文圈的有效補強來源目前是：
  - `bilibili`
  - `巴哈姆特`
  - `IG Reels`

## Strongest Reusable Shorts Clusters

- `個體值被砍 / 育成門檻大簡化`
  - 英日語圈都有短影音在做
  - 最適合當第一支預熱 Shorts
- `Mega 不是最強就會贏 / 構築比 Mega 更重要`
  - 日文 TikTok 題型成熟
  - 很適合翻成中文對戰 punch
- `HOME vs 招募 / 老玩家和新手起跑線不同`
  - 很有實用性，也很適合新手觀眾
- `新手第一隊先塞哪 3 格`
  - 日文 TikTok 很適合直接轉成中文實用短題
- `色違 / 新要素 / 意外情報`
  - 夠直觀，適合觸及到非深度競技觀眾

## Chinese Audience Fit

- 中文圈目前更吃：
  - `這作跟以前差在哪`
  - `這作有沒有比較好入坑`
  - `老玩家到底有沒有優勢`
  - `哪個新系統最值得注意`
- 純新聞更新型短片可以做，但不如 `一個 punch + 一個結論` 的題型有機會。

## Recommended First Shorts

- 第一支最推薦：
  - `寶可夢冠軍把個體值砍掉了`
- 第二支可接：
  - `老玩家跟新手，進《冠軍》的起跑線真的不同`
- 第三支可接：
  - `寶可夢冠軍不是 Mega 選最強就會贏`

## Caution

- `月卡值不值 / 免費但不是零成本` 在中文圈有量，但 cross-language Shorts 證據較弱，先當備用角度。
- 目前 YouTube Shorts 樣本薄，不要只看 YouTube；要固定用 `TikTok + IG Reels + 中文社群` 交叉驗證。
