# Pokemon Champions Doubles Meta Teams Memory

## Last Updated

- `2026-04-17`

## Summary

- 目前最值得記的雙打三大主流，不是單卡，而是三種系統：
  - `Mega花葉蒂控制平衡`
  - `Y噴晴天標準`
  - `雨隊 / 雨天空間`

## Strongest 3 Directions

### 1. Mega花葉蒂控制平衡

- 強度定位：
  - `目前最像版本答案`
- 強信號來源：
  - `Game8` 雙打 `SS`
  - `ビエラ` 率先 `2000`
  - `2100` 樣本持續出現
  - 巴哈雙打觀念文把 `花葉蒂-永恆之花` 視為泛濫核心
  - Reddit 已出現專門討論 `怎麼處理 Mega Floette`
- 常見支援件：
  - `Incineroar`
  - `Sableye`
  - `Maushold / Clefable / Sinistcha`
  - `Garchomp`
  - `Kingambit`

### 2. Y噴晴天標準

- 強度定位：
  - `最穩、最直觀、最適合新手理解`
- 強信號來源：
  - `Game8` 雙打 `Mega Charizard Y / Garchomp` 都在 `SS`
  - 巴哈 `Y噴雙打47勝大師球心得`
  - 巴哈 `Y噴軸M-1賽季大師級雙打隊伍分享`
  - B站 `版本答案，Y噴晴天隊`
  - 日文有 `晴れスタンダード` 與 `Y噴 + 妙蛙花`
- 常見支援件：
  - `Garchomp`
  - `Venusaur`
  - `Incineroar`
  - `Aerodactyl / Talonflame / Whimsicott`
  - `Kingambit`

### 3. 雨隊 / 雨天空間

- 強度定位：
  - `最像賽場式體系`
- 強信號來源：
  - `Game8`：`Archaludon S / Mega Blastoise S / Pelipper A`
  - JP YouTube：`ペリッパー＋ブリジュラス`
  - JP YouTube：`雨スイッチトリル`
  - 巴哈：`雨天空間隊20連勝上大師`
  - Reddit：`Rain Team Advice - Champions Doubles`
- 常見支援件：
  - `Pelipper`
  - `Archaludon`
  - `Farigiraf`
  - `Mega Blastoise`
  - `Incineroar`
  - `Basculegion♂ / Kingambit`

## Not Quite Top 3

- `Mega耿鬼滅歌 / Shadow Tag 控制`
  - 強，但目前跨平台重複度略低於前三
- `純重空間`
  - 有樣本，但目前更強的是 `重空間混合系`

## Reusable Video Angle

- 最好用的片型不是：

  - `現在最強雙打不是一隊，是這三套版本答案`

## Mega Floette Build Notes

- `Mega花葉蒂` 現在最值得記的，不只是它本體強，而是它有明確的幾種成形寫法
- `Game8` 同隊 TOP10：
  - `烈咬陸鯊`
  - `仆刀將軍`
  - `河馬獸`
  - `鋼鎧鴉`
  - `布里ジュラス`
  - `噴火龍`
  - `洗衣機`
  - `Hydreigon`
  - `盾劍怪`
  - `西獅海壬`
- 目前最穩的 5 種隊形理解：
  - `地鋼平衡`
  - `河馬展開型`
  - `龍鋼壓制型`
  - `雙 Mega 備選平衡`
  - `紅irection控制版`
- 如果之後要單拍 `Mega花葉蒂控制平衡`
  - 優先講 `河馬展開型`
  - 次選講 `地鋼平衡`
