# Tomodachi Life Game Memory

## Last Updated
- `2026-04-05`

## Core Positioning
- `Tomodachi Life: Living the Dream` 是 Nintendo 的 `Mii` 社交生活模擬作品。
- 官方定位不是硬核經營，也不是純建造，而是 `照顧一座充滿鬧劇、戀愛、關係變化和日常事件的小島`。
- 對台灣觀眾來說，最需要先回答的不是 `可不可愛`，而是 `這種觀察系人際沙盒適不適合你`。

## Official Facts
- Nintendo 美國商店頁目前顯示正式英文名：`Tomodachi Life: Living the Dream`
- 發售日：`2026-04-16`
- 支援語言包含：`Traditional Chinese`
- 官方描述重點：`silliness, drama, love, and other surprises`
- 可建立以自己、家人、朋友或原創角色為基礎的 `Mii`
- 可配置島上房屋、商店、餐廳與其他設施
- 有 `Mii News` 每日播報島上事件
- 目前官方也確認有 `free demo`
- demo 最多可建立 `3` 個 `Mii`
- demo 存檔可延續到正式版
- Nintendo JP FAQ 補充可直接確認：
  - 每位主機使用者各有自己的島與存檔
  - 每位使用者只有 `1` 筆存檔
  - `Mii` 最多可登錄 `70` 人
  - 可用主機內建 `Mii` 當基底建立住民
  - 不能直接從 `Miitopia` 等其他遊戲把角色整包搬進來
  - 本作裡做好的 `Mii` 也不能移到其他遊戲或主機 `Mii`
  - `Mii` 性別可選 `男性 / 女性 / 其他`
  - 戀愛對象可選 `男性 / 女性 / 其他`，可多選或不選
  - 不論性別，結婚的伴侶都可能迎來寶寶
  - Switch 2 沒有內容差異，但讀取更快，也支援 `GameChat` 畫面分享
  - `Mii` 建立與 `アイテム工房` 等部分場景支援觸控
  - 不支援 `Joy-Con 2` 滑鼠操作
  - 圖片影片可分享或直播，但無法使用部分主機直傳功能

## Useful Script Angle
- 最穩的 `買前必看` 切法：`別只看可愛`
- 第二切角：`不是動森 2.0`
- 適合喜歡看角色互動、自己捏熟人進去看鬧劇的玩家
- 如果想要的是高強度目標感、重資源效率或超深建造，買前要先校正期待

## Title Direction
- 最穩標題句型：`《Tomodachi Life 朋友收集 夢想生活》別只看可愛！買前要知道的 10 件事`
- 可替換備用：`不是動森 2.0`
- 可替換備用：`適不適合你`

- `它到底是什麼類型`

## Taiwan Community Hooks
- 台灣社群目前最在乎的不是 `畫風可不可愛`
- 而是：
  - `這到底在玩什麼`
  - `是不是動森 2.0`
  - `Mii 能捏多像`
  - `能不能分享 / 下載別人的 Mii`
  - `體驗版值不值得先玩`
  - `有沒有中文化`
- 最容易誤解的地方：
  - 把它當成 `動森 + 模擬市民`
  - 以為分享功能會像 `Miitopia` 或動森那樣方便
  - 以為它是完整線上社交遊戲
- 如果之後寫 `買前必看`，一定要把 `分享限制` 和 `demo / 中文化` 講進去
