## Stable Takeaways

- `Combo` 目前可穩講成：
  - `牌按 mana 遞增打，後面的牌倍率越來越高`
  - `Wild card 是 bridge，不是普通 0 費`
- `Buff` 要分成：
  - `跨 run 永久`
  - `本輪 run 持續`
  - `單場 battle`
- `Crawler / 角色卡`：
  - Trigger 會掉 duration
  - 最新首發文章明確寫它會跨 battle 持續到 activations 用完
- `Blacksmith / Jeweler`：
  - Jeweler 管 gem 掉落方向
  - Blacksmith 管永久打洞
- `Power-up priority`：
  - 最新首發攻略主流已偏向 `Greed -> Might`
  - 但今日社群仍有 `Armor` 先點派
  - 之後寫稿要講成 `兩條路線`
