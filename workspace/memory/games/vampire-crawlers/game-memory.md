## Core Positioning
- `Vampire Crawlers` 不是單純把《Vampire Survivors》換皮成卡牌。
- 它對台灣觀眾最值得先回答的問題是：
  - `這款到底有沒有把原本的爽感成功轉成卡牌 / 地城玩法`
  - `新手第一天要先懂什麼，才不會覺得它不爽`
- 對 `玫玫物語` 來說，這款最穩的內容方向不是純實況，而是：
  - `買前 / 首發值不值得買`
  - `新手開局`
  - `combo / gem / 進化武器`
