# 2026-04-22 Newbie Tip Signal

## 這輪研究最重要的結論
- 這款的 `新手攻略`，核心不是先列武器名單。
- 最穩的片子骨架應該先從：
  - `combo 規則`
  - `抽牌 / mana / 手牌`
  - `地圖與 shovel`
  - `power-up 投資`
  - `小而精牌組`
  - `武器進化`
  - `寶石用途`
  開始。

## 為什麼
- 目前真正高訊號的字幕、論壇和文章正文，反覆在講的不是：
  - `哪隻角色最帥`
  - `哪把武器數字最大`
- 而是：
  - `combo 怎麼接`
  - `wild 怎麼用`
  - `前期資源怎麼分`
  - `什麼時候該留怪刷 buff`
  - `什麼牌不要亂撿`

## 目前最穩的新手技巧池
- 開局先懂 combo
- 同費 / 跳費 / wild 規則
- 抽牌 + mana + 手牌優先
- 角色按色卡 synergy 選
- coins 先投 health / growth / armor / recovery / reroll
- 每一輪看 unlocks
- 規劃路線，不要急著踩 shovel
- elite 後打，問號和資源先拿
- 分清永久 buff 和單戰 buff
- safe 局留怪刷 buff
- deck 不要太胖
- crawler 盡量延長
- 武器進化前不要先鑲 gem
- gem 要服務 build

## 暫時不要推太前面的點
- `最強角色`
- `最強 build`
- `秘密角色`
- `紫死神`
- `無限 combo`

## 判斷理由
- 這些題目不是不能講，而是對 `第一支新手 10 技巧` 來說太後段。
- 先把玩家最常卡的理解門檻講通，影片體感會比硬塞 meta 更像 `玫玫物語`。
