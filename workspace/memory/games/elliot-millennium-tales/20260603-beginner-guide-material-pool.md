# 冒險家艾略特的千年奇譚｜正式版新手攻略待研究素材池

日期：2026-06-03  
來源：使用者本輪整理與已鎖定系列定位  
用途：之後進入 `正式上線後的新手攻略` 研究時，作為素材池與查證清單。這份不是深度研究包，也不是章節大綱或正式稿。

## 本輪工作邊界

- 本片是艾略特系列第 3 類內容：`正式上線後的新手攻略`。
- 已鎖定前兩類不可混用：`買之前必看`、`體驗版先做`。
- 新手攻略方向：偏實用，處理玩家剛玩時容易漏掉、容易誤會、能少走彎路的系統與操作。
- 本輪只整理素材，不開始深度研究、不查外部資料、不強制訂章節、不收斂成口播稿。
- 等使用者之後明確說 `開始研究`，或給定章節 / 研究方向後，再依鎖定章節做深度研究。

## 2026-06-07 使用者本輪指定章節草稿

狀態：使用者口述整理。這裡只記錄使用者指定的章節方向與要講的點；尚未做正式深度研究、查證正式版資料或改成可朗讀稿。

### 第一章：開局設定

- 章節方向：開局設定要先講 `難度選擇` 和 `搭檔對話頻率`。
- 難度選擇要講：
  - 不同難度的差異。
  - 新手到底該選什麼難度。
  - 之後正式研究要補清楚：各難度差異、適合對象、是否影響挑戰證明 / 成就 / 獎勵。
- 搭檔對話頻率要講：
  - 搭檔對話可以調高或調低。
  - 調高的好處是遊戲中有時會透過左下角搭檔語音提示玩家，例如：
    - 附近是否還有寶箱。
    - 地宮 / 迷宮裡是否還有未探索內容。
    - 某些解謎或探索要素，例如需要用炸彈炸開。
  - 如果玩家覺得太吵，或已經不需要提示，可以把搭檔對話調低。
  - 探索按鍵 / 探索提示也可能會告訴玩家附近是否已經沒有東西可探索。

### 第二章：掉落獎勵

- 章節方向：講掉落獎勵規則，以及不同掉落獎勵星級 / 狀態下的差距。
- 要講出：
  - 掉落獎勵怎麼出。
  - 掉落獎勵的規則。
  - 不同掉落狀態帶來的差異。
  - 打完 Boss 後尤其要注意，Boss 掉落的魔石碎片會散在很大範圍。
  - 如果不趕快撿，掉落物會在原地消失。

### 第三章：魔石系統

- 章節方向：講魔石系統規則，以及它為什麼是主要戰力來源。
- 要講出：
  - 魔石的基本規則。
  - 什麼時候要把魔石碎片拿去兌換 / 生成魔石。
  - 不同等級或階級升上去有什麼好處。
  - 不同武器都要注意魔石配置。
  - 魔石箱 / COST 容量要記得升級，因為容量提高後，武器可以裝備更多魔石。
  - 魔石與魔石箱升級是主要提升戰力的來源之一。

### 待放入章節：地圖與寶箱

- 章節方向：看地圖拿寶箱。
- 要講出：
  - 打開地圖後，移到區域 / 遺跡 / 迷宮上，通常會顯示該地區寶箱剩餘數量。
  - 在遺跡裡打開地圖，上方也會顯示寶箱數量。
  - 正常來說，盡量把所有寶箱拿乾淨。
  - 前面提到的搭檔有時候會提醒玩家這個地宮還有寶箱沒有探索。
  - 如果使用探索按鍵，也可能提示附近是否沒有東西可探索。

### 使用者追加可講章節：商店道具與配件

- 章節方向：介紹道具商店裡的消耗品、容器與配件，重點不是全買，而是講新手哪些真的有必要買。
- 要講出：
  - 藥水與玻璃管非常重要，基本上一定要買。
  - 需要講清楚：買藥水可以在什麼情況下使用，例如 Boss 戰、探索中補血、失誤後保命。
  - 玻璃管是藥水容器，沒有容器就不能只靠買藥水解決補血問題；正式稿前要確認繁中 UI 名稱與規則。
  - 商店裡的配件要分成「新手必要 / 視情況買 / 之後再買」。
  - 要特別講披風類配件，以及可以降低或處理地形傷害的商店道具 / 配件。
  - 這章要回答觀眾真正會問的問題：這些商店道具到底有沒有必要買，還是先省錢升魔石箱。

### 使用者追加可講章節：戰鬥技巧與武器選擇

- 章節方向：專門講戰鬥技巧、武器選擇，以及精準防禦。
- 要講出：
  - 不同武器手感與用途不同，新手要知道不是只看攻擊力。
  - 劍、槍、弓、炸彈、鎚子、鎖鎌等武器要依攻擊範圍、距離、命中穩定度、解謎用途來介紹。
  - 直線型武器需要對位，範圍型武器比較適合新手穩定命中。
  - 可以精準防禦；這不是只有高手才用，而是高難度或 Boss 戰會明顯影響生存。
  - 之後研究要補：精準防禦觸發時機、盾耐久、是否有配件能放寬判定、哪些武器適合搭配防禦反擊。

## 2026-06-07 上市後影片預計題修正

- 使用者修正：目前上市後先預計出兩篇，分別是 `高效率刷魔石碎片` 與 `魔石 Full / 魔石箱滿級養成`。
- `肅殺模式 / 最高難度` 先改回候選題，不是本輪使用者指定一定要先出的片；正式名稱、解鎖條件與需求要等上市後實機確認。
- 搭檔提示相關題目要補充：提示太高雖然能少漏寶箱、少漏解謎線索，但也可能破壞玩家自己解謎的樂趣。
- 攻略理念要講玩家分流：想自己探索、保留解謎樂趣的玩家可以調低；想省時間、少漏寶箱或不想卡關的玩家可以調高。

## 分類口徑

- `已確認`：目前可當成高信心素材保存，但正式稿前仍要核對繁中 UI、正式版版本與是否沿用。
- `使用者體驗觀察`：來自使用者實機感受、遊玩流程或暫時判斷；可作為研究起點，不能直接升級成普遍結論。
- `需要正式研究查證`：機制、數字、正式版解法、取得方式、是否會錯過、是否沿用體驗版刷法等，之後研究時必查。

## 設定與操作

### 已確認

- 設定裡可以調整搭檔的對話頻率；如果覺得太吵，可以調低。
- 自動過場可以關掉。
- 可以裝備多種武器；後續攻略需要介紹武器切換與不同武器用途。

### 使用者體驗觀察

- 右下角小地圖有時候黑黑的、很難看；建議直接打開完整地圖看地形，比較好走。
- 攻擊是八方向，不是自由瞄準，所以直線型武器比較容易打空。
- 劍目前玩起來最好用，因為是扇形攻擊，命中比較穩。
- 長槍、弓箭這種直線武器，打斜向敵人要瞄準，不然會空揮。

### 需要正式研究查證

- 小地圖上哪些圖示最重要。
- 武器切換的正式 UI、前期推薦攜帶組合、各武器適合的用途。
- 藥水 / 空玻璃管 / 回復道具的運作方式，以及新手是否要優先購買。

## 地圖與探索

### 已確認

- 地圖上會顯示該迷宮還剩下多少寶箱沒開；這代表該區域還有寶箱沒拿。
- 進遺跡或洞窟時，要打開地圖確認寶箱數量，不然很容易漏。
- 有些寶箱藏在密道、裂牆、石頭後面。
- 裂痕牆壁、大石頭之類的障礙，需要後面拿到炸彈或特定能力再回來處理。
- 路上 NPC 對話有時候會獲得物品，新手要提醒不要只衝主線。
- 生命神殿很像薩爾達神殿，有挑戰型和戰鬥型，完成後可以拿生命碎片。
- 生命值不是靠升級，而是靠蒐集生命碎片提升上限。

### 使用者體驗觀察

- 護佑時代一開始有大石頭、牆壁擋住的地方；正式版要研究實際解法。
- `拿到新能力記得回頭探索` 可以成為新手攻略素材，因為炸彈取得後可回頭處理裂牆和石頭。

### 需要正式研究查證

- 護佑時代大石頭、牆壁、裂牆的正式版解法與回頭時機。
- 極光幕怎麼解，不要先講死。
- 地上的大洞怎麼過，不要先講死。
- 炸彈正式取得位置與時機。
- 正式版容易錯過的寶箱、生命碎片、支線與 NPC 物品。

## 掉落與刷資源

### 已確認

- 地上掉落的金幣、魔石碎片不會自動進背包。
- 掉落物時間到看起來會消失，而且不會自己收進背包，所以要記得撿。
- 尤其是打完 Boss 或大量掉落時，要先把地上的東西撿乾淨。
- 掉落獎勵是連續無傷擊殺怪物累積。
- 5 殺會到 1 星，15 殺會到 2 星，30 殺會到 3 星。
- 殺掉魔物後身上會出現目前幾 `CHAIN`。
- 受到傷害會中斷掉落獎勵。
- 蓮藕護符可以讓掉落獎勵不會因為受到傷害中斷一次，刷魔石 / 刷錢時很重要。

### 使用者體驗觀察

- 刷資源時，新手最容易虧的是打完怪或 Boss 後沒有先把地上的掉落物撿乾淨。
- 掉落獎勵與保 chain 道具，比單純硬刷怪更像這款遊戲的前期效率核心。

### 需要正式研究查證

- `雙倍掉落` 到底是什麼。
- 正式版怎麼快速刷錢和魔石。
- 蓮藕護符正式版取得方式、限制、是否能重複保護 chain。
- 正式版是否還能沿用體驗版刷法。
- 效率數字必須標明來源層級，不可憑印象保證收益。

## 魔石系統

### 已確認

- 魔石可以讓武器變強，也可以增加效果。
- 魔石不是單純加數值，可能會改變武器效果、提升蓄力、暴擊、擊飛等。
- 魔石有 `COST`，要在有限容量裡取捨。

### 使用者體驗觀察

- 魔石系統有機會成為新手攻略重點，因為它牽涉前期武器手感、刷怪效率與資源配置。

### 需要正式研究查證

- 魔石碎片、魔石合成、魔石生成階級、魔石箱升級的正式版流程與前期優先順序。
- 新手先刷什麼魔石。
- 哪些魔石適合前期。
- 哪些只是體驗版可先準備，不一定該放進正式版新手攻略主軸。
- 魔石前期怎麼刷最有效率。

## 難度

### 已確認

- 難度會影響受到傷害、敵人體力、敵人數量、接觸敵人是否受傷、道具價格、復活費用、保存點恢復限制。
- 簡單難度道具比較便宜，想先買重要道具、補道具或刷資源，可以考慮調低難度。
- 普通難度下單純碰到怪不會受傷；困難以上接觸敵人會扣血，刷連殺比較容易斷。

### 使用者體驗觀察

- 新手攻略應該講清楚兩種玩家分流：想挑戰極難的人不要亂調；只想舒服探索和刷資源的人可以善用難度調整。

### 需要正式研究查證

- 極難正式版獎勵 / 成就條件。
- 最高難度是否只能遊戲一開始設定。
- 中途調低難度是否會失去極難相關成就 / 獎勵。
- 正式版難度差異是否和體驗版截圖 / 觀察完全一致。

## 任務與時效

### 已確認

- 有些支線任務有時效性，主線推過後可能不能接。
- 任務清單可以看到目前有哪些支線可以接。
- 新手攻略要提醒：推主線前先看任務清單、先接支線。

### 使用者體驗觀察

- 這個題材適合放在 `不要只衝主線` 的實用提醒裡，但正式研究前不能擴張成全支線攻略。

### 需要正式研究查證

- 哪些任務會錯過。
- 哪些支線值得先做。
- 任務清單顯示規則、錯過條件、是否受主線進度或時代影響。

## 體驗版 / 先做內容可轉進新手攻略的素材

### 已確認

- 體驗版完整蒐集回報：生命最多可以到 8 格，身上還會剩 3 個生命碎片。這屬於體驗版先做內容；如果新手攻略要講 `不要只衝主線`，可以拿來當例子。
- 護佑時代北之洞窟有需要輕盈披風才能拿到的隱藏寶箱。
- 輕盈披風普通難度 1000，簡單難度 500。
- 北之洞窟可拿到 4 星魔石 `炎舞`，但偏體驗版先做，放不放進新手攻略要看章節。
- 空玻璃管、輕盈披風、滑翔相關道具，可能是前期值得先買的東西。
- 炸彈拿到後可以回頭處理裂牆和石頭。

### 使用者體驗觀察

- 體驗版素材可以當例子或補充，不應該把正式版新手攻略變成 `體驗版先做` 的重講。

### 需要正式研究查證

- 正式版是否仍可沿用體驗版刷法、道具價格、寶箱位置與前期購買優先順序。
- 體驗版可取得內容哪些在正式版變成新手期仍重要。
- 體驗版素材放進新手攻略時，要先確認章節用途，不能自動升成主軸。
