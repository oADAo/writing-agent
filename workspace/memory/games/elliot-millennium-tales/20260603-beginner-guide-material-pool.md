用途：之後進入 `正式上線後的新手攻略` 研究時，作為素材池與查證清單。這份不是深度研究包，也不是章節大綱或正式稿。

- 新手攻略方向：偏實用，處理玩家剛玩時容易漏掉、容易誤會、能少走彎路的系統與操作。

## 2026-06-07 使用者本輪指定章節草稿

狀態：使用者口述整理。這裡只記錄使用者指定的章節方向與要講的點；尚未做正式深度研究、查證正式版資料或改成可朗讀稿。

### 第一章：開局設定

- 章節方向：開局設定要先講 `難度選擇` 和 `搭檔對話頻率`。
- 難度選擇要講：
  - 不同難度的差異。
  - 新手到底該選什麼難度。
  - 之後正式研究要補清楚：各難度差異、適合對象、是否影響挑戰證明 / 成就 / 獎勵。
- 搭檔對話頻率要講：
  - 搭檔對話可以調高或調低。
  - 調高的好處是遊戲中有時會透過左下角搭檔語音提示玩家，例如：
    - 附近是否還有寶箱。
    - 地宮 / 迷宮裡是否還有未探索內容。
    - 某些解謎或探索要素，例如需要用炸彈炸開。
  - 如果玩家覺得太吵，或已經不需要提示，可以把搭檔對話調低。
  - 使用者實機補充：自己目前是開低，但遇到解謎卡住後發現妖精提示確實有用，接下來會試著開高。正式稿可建議新手先開高，熟悉後再依喜好調低。
  - 如果一時間沒看清妖精說什麼，先停下來回看 / 回想前一句提示，或重新接近門、路、機關再觸發一次提示；不要因為漏看一句就硬解十分鐘。
  - 探索按鍵 / 探索提示也可能會告訴玩家附近是否已經沒有東西可探索。

### 第二章：掉落獎勵

- 章節方向：講掉落獎勵規則，以及不同掉落獎勵星級 / 狀態下的差距。
- 要講出：
  - 掉落獎勵怎麼出。
  - 掉落獎勵的規則。
  - 不同掉落狀態帶來的差異。
  - 打完 Boss 後尤其要注意，Boss 掉落的魔石碎片會散在很大範圍。
  - 如果不趕快撿，掉落物會在原地消失。

### 第三章：魔石系統

- 章節方向：講魔石系統規則，以及它為什麼是主要戰力來源。
- 要講出：
  - 魔石的基本規則。
  - 什麼時候要把魔石碎片拿去兌換 / 生成魔石。
  - 不同等級或階級升上去有什麼好處。
  - 不同武器都要注意魔石配置。
  - 魔石箱 / COST 容量要記得升級，因為容量提高後，武器可以裝備更多魔石。
  - 魔石與魔石箱升級是主要提升戰力的來源之一。

### 待放入章節：地圖與寶箱

- 章節方向：看地圖拿寶箱。
- 要講出：
  - 打開地圖後，移到區域 / 遺跡 / 迷宮上，通常會顯示該地區寶箱剩餘數量。
  - 在遺跡裡打開地圖，上方也會顯示寶箱數量。
  - 正常來說，盡量把所有寶箱拿乾淨。
  - 前面提到的搭檔有時候會提醒玩家這個地宮還有寶箱沒有探索。
  - 如果使用探索按鍵，也可能提示附近是否沒有東西可探索。

### 使用者追加可講章節：商店道具與配件

- 章節方向：介紹道具商店裡的消耗品、容器與配件，重點不是全買，而是講新手哪些真的有必要買。
- 要講出：
  - 藥水與玻璃管非常重要，基本上一定要買。
  - 需要講清楚：買藥水可以在什麼情況下使用，例如 Boss 戰、探索中補血、失誤後保命。
  - 玻璃管是藥水容器，沒有容器就不能只靠買藥水解決補血問題；正式稿前要確認繁中 UI 名稱與規則。
  - 前期應該早點買的商品要整理成清單：空玻璃管、回復藥、輕盈披風、可能降低地形傷害或提升探索容錯的配件。
  - 要補一段「哪裡可以補滿血」：紅色冒險道標 / 存檔點、能傳送回血的地點、Boss 前推薦補狀態路線。
  - 商店裡的配件要分成「新手必要 / 視情況買 / 之後再買」。
  - 要特別講披風類配件，以及可以降低或處理地形傷害的商店道具 / 配件。
  - 這章要回答觀眾真正會問的問題：這些商店道具到底有沒有必要買，還是先省錢升魔石箱。

### 使用者追加可講章節：妖精能力與提示判斷

- 章節方向：獨立介紹妖精能力，不只當成戰鬥跟班，而是當成戰鬥、解謎、探索回頭、提示判斷的系統。
- 要講出：
  - 妖精可設定成 `突擊`，讓她自己攻擊與行動；這可能是新手保命、拉怪、保 Drop Bonus 的重要設定。
  - 妖精能用在復活、點火、吸引敵人 / 掉落物、傳送、分身、踩開關或遠距離解謎等場景。
  - 妖精提示「現在無法打開這個門」或「現在無法過去」時，要提醒觀眾不要硬撞；多半代表缺能力、缺主線進度或要換時代處理。
  - 使用者實機案例：解到某個地方時，妖精提示現在解不了；因此直接離開，不硬解。這可以做成「聽到這種提示就先走」的避坑點。
  - 這章要搭配「拿到新能力記得回頭探索」，因為很多門、路徑、寶箱不是當下能處理。
  - 正式稿前要補：妖精各模式的正式 UI 名稱、設定位置、各魔法取得時機與對應解謎用途。

### 使用者追加可講章節：戰鬥技巧與武器選擇

- 章節方向：專門講戰鬥技巧、武器選擇，以及精準防禦。
- 要講出：
  - 不同武器手感與用途不同，新手要知道不是只看攻擊力。
  - 劍、槍、弓、炸彈、鎚子、鎖鎌等武器要依攻擊範圍、距離、命中穩定度、解謎用途來介紹。
  - 直線型武器需要對位，範圍型武器比較適合新手穩定命中。
  - 可以精準防禦；這不是只有高手才用，而是高難度或 Boss 戰會明顯影響生存。
  - 之後研究要補：精準防禦觸發時機、盾耐久、是否有配件能放寬判定、哪些武器適合搭配防禦反擊。

## 2026-06-18 使用者鎖定 7 章順序

狀態：已鎖定。後續研究、篩選、Excel 勾選、章節補強與寫稿 handoff 都要保留這 7 章和順序；不可擅自重排、合併、替換或改成 10 章。

1. **開局設定**
   - 難度選擇。
   - 搭檔對話頻率。
   - 新手先開高，因為菲伊會提示卡關 / 解謎 / 不能過的地方。

2. **妖精能力與提示判斷**
   - 突擊設定。
   - 自動攻擊 / 保命 / 拉怪。
   - 復活、點火、吸引、傳送、分身、解謎。
   - 菲伊說現在不能開 / 不能過，就不要硬解，先推主線或拿能力。

3. **掉落獎勵**
   - Drop Bonus 規則。
   - 星級 / 階段差異。
   - 被打會斷、保 chain、蓮根護符。
   - 刷錢不要只講蓮根 / Drop Bonus，要補「寶石怪打了會大量掉金幣，但出現條件未知」。

4. **魔石系統**
   - 魔石規則。
   - 魔石為什麼是主要戰力來源。
   - 魔石主要的刷法是什麼。
   - 魔石箱 / COST 容量優先升級。
   - 魔石碎片就是拿來隨機生成魔石；身上有多的就能拿去生成魔石；魔石可以說是最主要的戰力。

5. **地圖與寶箱**
   - 看地圖拿寶箱。
   - 寶箱數量 / 探索提示。
   - 有些東西現在拿不到，要之後拿能力回來。
   - 跨時代解謎，過去點火會影響現代。

6. **商店道具與配件**
   - 道具商店裡哪些真的該買。
   - 空玻璃管、回復藥、輕盈 / 浮空披風、探索容錯配件。
   - 到每個時代的新城鎮先買空玻璃管、補炸彈、補弓箭、看新武器 / 配件 / 支線。
   - 簡單難度商品半價。
   - 輕盈 / 浮空披風在前期探索有幫助，但後面有妖精的傳送技能後，就不一定要用披風。
   - 想擴充容量時，要看炸彈袋、箭筒、魔石箱 / COST 容量與相關支線。

7. **戰鬥技巧與武器選擇**
   - 精準防禦。
   - 劍、槍、弓、炸彈、迴旋鏢等武器差異。
   - 不同武器不要只看攻擊力，要看命中、距離、手感、魔石搭配。
   - 使用者目前覺得長槍好用，魔石配置是三頭龍 + 魔法長槍。
   - 其他好用配置可留給觀眾留言補充，不要講成唯一最強。

## 設定與操作

### 已確認

- 設定裡可以調整搭檔的對話頻率；如果覺得太吵，可以調低。
- 自動過場可以關掉。
- 可以裝備多種武器；後續攻略需要介紹武器切換與不同武器用途。

### 使用者體驗觀察

- 右下角小地圖有時候黑黑的、很難看；建議直接打開完整地圖看地形，比較好走。
- 攻擊是八方向，不是自由瞄準，所以直線型武器比較容易打空。
- 早期體感曾偏向劍，因為是扇形攻擊、命中比較穩；2026-06-18 後續校正：使用者目前覺得長槍好用，魔石配置是三頭龍 + 魔法長槍；其他好用配置可留給觀眾留言補充。
- 長槍、弓箭這種直線武器，打斜向敵人要瞄準，不然會空揮。
- 妖精正常可以設定成 `突擊`，讓她自己攻擊與行動；這可能是新手保命、拉怪、保 Drop Bonus 的重要設定。
- 妖精能力值得獨立成章，因為它同時影響戰鬥、解謎、探索回頭、過不了門時的提示判斷。

### 需要正式研究查證

- 小地圖上哪些圖示最重要。
- 武器切換的正式 UI、前期推薦攜帶組合、各武器適合的用途。
- 藥水 / 空玻璃管 / 回復道具的運作方式，以及新手是否要優先購買。
- 妖精 `突擊` 的正式 UI 名稱、設定位置、是否有攻擊 / 防守 / 跟隨等模式差異。
- 妖精提示文字的完整分類：無法開門、無法過去、需要能力、需要主線進度、需要特定時代解謎。

## 地圖與探索

### 已確認

- 地圖上會顯示該迷宮還剩下多少寶箱沒開；這代表該區域還有寶箱沒拿。
- 進遺跡或洞窟時，要打開地圖確認寶箱數量，不然很容易漏。
- 有些寶箱藏在密道、裂牆、石頭後面。
- 裂痕牆壁、大石頭之類的障礙，需要後面拿到炸彈或特定能力再回來處理。
- 路上 NPC 對話有時候會獲得物品，新手要提醒不要只衝主線。
- 生命神殿很像薩爾達神殿，有挑戰型和戰鬥型，完成後可以拿生命碎片。
- 生命值不是靠升級，而是靠蒐集生命碎片提升上限。

### 使用者體驗觀察

- 護佑時代一開始有大石頭、牆壁擋住的地方；正式版要研究實際解法。
- `拿到新能力記得回頭探索` 可以成為新手攻略素材，因為炸彈取得後可回頭處理裂牆和石頭。

- 菲伊 / 妖精提示「現在無法打開這個門」或「現在無法過去」時，不一定是玩家漏看；通常可先當作缺能力或缺主線進度，繼續主線後再回來。

## 掉落與刷資源

### 已確認

- 地上掉落的金幣、魔石碎片不會自動進背包。
- 掉落物時間到看起來會消失，而且不會自己收進背包，所以要記得撿。
- 尤其是打完 Boss 或大量掉落時，要先把地上的東西撿乾淨。
- 掉落獎勵是連續無傷擊殺怪物累積。
- 5 殺會到 1 星，15 殺會到 2 星，30 殺會到 3 星。
- 殺掉魔物後身上會出現目前幾 `CHAIN`。
- 受到傷害會中斷掉落獎勵。
- 蓮藕護符可以讓掉落獎勵不會因為受到傷害中斷一次，刷魔石 / 刷錢時很重要。

### 使用者體驗觀察

- 刷資源時，新手最容易虧的是打完怪或 Boss 後沒有先把地上的掉落物撿乾淨。
- 掉落獎勵與保 chain 道具，比單純硬刷怪更像這款遊戲的前期效率核心。
- 使用者後續校正：刷錢重點不要過度放在蓮藕護符 / Drop Bonus 獎勵；如果遇到「寶石怪」，打了會大量掉金幣，實用優先級更高。出現條件目前未知，正式稿只能講成「遇到一定要把握」或「高價值遭遇」，不能講成穩定刷法。

### 需要正式研究查證

## 難度

### 已確認
