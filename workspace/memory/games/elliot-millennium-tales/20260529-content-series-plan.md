# 冒險家艾略特的千年奇譚｜內容系列規劃

用途：後續研究與寫稿歸類，不是正式稿。

## 已鎖定系列順序

### 1. 買之前必看

- 目標：幫觀眾判斷值不值得買、要不要先玩 Demo、適不適合首發。
- 重點：遊戲本身、好不好玩、玩法定位、劇情演出、戰鬥、武器、魔石、探索、難度、Demo 能測什麼。
- 不要塞太多攻略細節；體驗版刷法只當加料，不要搶走主軸。
- 目前主研究包：`workspace/deliverables/research/20260528-elliot-buy-before-pre-research.md`

### 2. 體驗版先做

- 目標：正式上市前，告訴觀眾 Demo 先做哪些事最有用。
- 重點：不是刷等級，而是刷魔石、魔石碎片、錢、生命碎片、支線、寶箱、可取得 / 暫時拿不到的內容。
- 可用角度：`體驗版上市前先做 5 件事`、`正式版前不要漏掉這些獎勵`、`哪些值得農，哪些不用農到厭世`。
- 目前研究筆記：`workspace/memory/games/elliot-millennium-tales/20260529-new-player-guide-notes.md`

### 3. 正式上線後的新手攻略

- 目標：正式版上線後，做真正的新手開局 / 前期必做攻略。
- 重點：設定、搭檔對話頻率、自動過場、大地圖、小地圖資訊、掉落物是否自動入包、遺跡寶箱數量、NPC 對話獲得物品、雙倍掉落、裝備多種武器、藥水。
- 正式版後要重新查證：版本更新、UI 名稱、體驗版和正式版是否有差異、玩家實測刷法是否被改。
- 不要直接把體驗版攻略照搬成正式版攻略。
