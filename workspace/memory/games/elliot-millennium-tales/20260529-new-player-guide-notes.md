### 地圖寶箱數量：已確認方向

- 使用者實機確認：地圖上會顯示該迷宮還剩下多少寶箱沒開。
- 攻略用途：這可以做成新手攻略重點，提醒觀眾進迷宮時打開地圖看寶箱剩餘數，不要只看右下角小地圖。
- 來源補強：Game Tier 攻略也提到洞窟地圖會以類似 `寶箱 x0` 的方式顯示殘數，應探索到沒有寶箱為止。

### 掉落增加 / Drop Bonus

- 使用者實機確認：不受傷連續殺怪會讓掉落不斷增加。
- 使用者實機觀察：5 殺為 1 星，15 殺為 2 星，之後階段尚未確認。
- 使用者實機觀察：連擊時，殺掉魔物後角色身上會顯示目前是幾 `CHAIN`，可用來確認連殺是否還在。
- 使用者實機確認：`蓮藕護符` 可以讓掉落獎勵不會因為受到傷害而中斷一次。這是刷錢 / 刷魔石時很重要的保險道具或配件。
- 目前待查：3 星 / 最高星數需要幾殺、是否到 30 chain 封頂、蓮藕護符取得方式、是否消耗、是否有冷卻、能否多次保 chain、被地形傷害 / 掉落傷害是否會中斷。
- 來源補強：Game Tier 攻略提到連續擊殺會觸發 Drop Bonus / chain，受傷會中斷，最大 30 chain；低難度更容易維持 chain。

### 支線任務時效性與任務清單

- 使用者實機觀察：有些任務有時效性，主線推過之後就不能接。
- 使用者實機觀察：從任務清單可以看到現在有哪些支線可以接。
- 攻略用途：新手攻略要提醒觀眾主線推太快前，先開任務清單確認能接的支線；體驗版先做片可以列 `先清支線再推主線`。
- 待查：哪些支線會錯過、錯過條件、任務清單是否只顯示可接還是也顯示已接 / 可追蹤任務。

### 買前補充：生命神殿

- 使用者買前觀察：有類似薩爾達神殿的地方，叫做生命神殿。
- 使用者觀察：生命神殿裡可能有一個謎題或挑戰，解完後可以拿到生命碎片。
- 使用者實機補充：生命神殿真的很像薩爾達小神殿，會分成不同挑戰型態。
- 使用者實機補充：`鍛鍊之間` 偏操作挑戰，裡面會放一堆陷阱，測玩家走位與操作。
- 使用者實機補充：`力量試煉` 偏戰鬥挑戰，裡面會放一堆怪物，要殺完全部怪物。
- 買前稿用途：放在探索 / 成長章，說明它的探索不是只有寶箱，也有類似小神殿的挑戰、戰鬥房與生命上限成長。

### 擋路大石頭 / 牆壁：目前整理

- 有裂痕的牆、特定岩石：目前來源指向用 `炸彈` 處理。GNN 試玩報導提到炸彈可破壞特定岩石與有裂縫的牆；Inside Games / ASCII 也提到體驗版能用炸彈破壞裂牆。
- 木樁 / 釘樁類阻擋：來源指向 `鎚子` 或後續機關。GNN 試玩報導提到鎚子可以敲平阻擋去路的木樁；日文體驗版攻略則把 `打樁機關` 列成可能要等正式版 / 未開放手段。
- 點火 / 冰 / 火把機關：可能需要菲伊的魔法，例如點燃 / チャッカ；沒拿到能力前先標記，不要硬找。
- 鑰匙 / 妖精能力 / 未開放區：日文體驗版攻略明確提醒，有些寶箱或路線在體驗版目前拿不到，不需要硬卡。
- 回答原則：看到擋路先判斷外觀。裂牆 / 特定石頭試炸彈；木樁等鎚子；火把等菲伊點燃；寶箱數剩 1 但怎麼都過不去時，很可能是正式版能力或鑰匙。

### 機關：極光幕與地上的大洞

- 使用者提問：有一種 `極光幕` 擋住路，暫時不知道怎麼解。
- 目前整理：如果是王國周邊或野外邊界那種彩色光幕，來源更像是劇情中的 `加護魔法 / 結界`，不是炸彈、武器或一般機關可硬解；先推主線、看任務旗標或從其他入口繞路。若在迷宮房間內出現，先檢查是否需要清怪、踩機關、拿鑰匙，或屬於體驗版未開放路線。
- 使用者提問：地上有個大洞，不知道怎麼解。
- 目前整理 1：如果是會把人吸進去的洞，AppMedia 體驗版攻略寫法是把 `炸彈` 丟進洞裡破壞，靠近前先處理，不然會被吸進去受傷。
- 目前整理 2：如果是單純跳不過的坑 / 斷路，官方與攻略站都提到夥伴能力可用於探索；常見解法是用夥伴位置 `傳送 / ワープ` 到對岸，或用高速移動過會掉落的地板。還沒拿到對應能力時先標記，不要硬跳。
- 拍片用途：新手攻略可做 `看到過不去的路先分四種：炸彈洞、裂牆、夥伴傳送、劇情結界`。

### 快速刷錢與魔石：目前整理

- 核心機制：維持 Drop Bonus。使用者已確認不受傷連殺會增加掉落，攻略站也提到受傷會中斷 chain。
- 難度選擇：Game Tier 建議低難度更容易維持連殺，因為敵人弱、比較容易一擊處理；巴哈玩家則回報迷霧遺跡在困難難度刷碎片和錢效率較好，極難怪更多但走位和一槍擊殺穩定性變差。
- 路線 1：迷霧遺跡。巴哈玩家用困難難度在迷霧遺跡刷魔石碎片和錢，主力裝備為長槍和弓。
- 路線 2：白色區域北部 / 北洞。日文攻略提到北洞附近有掉 10 金左右的敵人，長槍突進相性好，能一邊刷錢一邊刷魔石。
- 路線 3：沙漠洞窟。日文攻略提到後半金策效率也高，但敵人配置較麻煩、比較容易被打斷 chain。
- 推薦裝備：長槍主力，搭配蓄力縮短、蓄力移動速度、滿血敵人增傷、突進強化；周回時最好裝備 chain 維持或掉落相關護符。
- 物件破壞：日文攻略提醒，部分壺 / 物件也會出魔石碎片，周回時不要只打怪。

### 設定：搭檔對話頻率與自動過場

- 使用者實機觀察：設定裡可以調整搭檔的對話頻率。
- 使用者判斷：如果覺得搭檔太吵，可以把對話頻率調低。
- 使用者實機觀察：自動過場也可以關掉。
- 後續要查：正式 UI 名稱、最低頻率實際效果、是否只影響閒聊或也影響提示、是否建議新手一開始就調低。

### 地圖：右下角小地圖、大地圖與路線判讀

- 使用者實機觀察：右下角地圖黑黑的很難看。
- 使用者建議：可以打開地圖直接看全部地形，比較好走。
- 使用者實機觀察：小地圖上有很多資訊可以看。
- 後續要查：小地圖 / 大地圖各自顯示哪些圖示，是否有寶箱、NPC、出入口、主線目標、支線或地形高低差提示。
- 後續攻略用途：可以做一段 `找不到路先開大地圖，不要只看右下角小地圖`。

### 掉落物：金幣和魔石碎片是否需要手動撿

- 使用者實機確認：地上掉落物不撿就會消失，不會自己進背包。
- 使用者提醒：打完 Boss 掉落一堆時要趕快撿，否則可能直接消失。
- 來源補強：AppMedia 的菲伊說明提到可操作菲伊去回收遠處的金錢與道具；也就是掉落物需要實際被回收，不是無條件自動入包。
- 後續要查：不同掉落物消失前大約多久、Boss 掉落物是否和一般怪相同、收集範圍配件是否能降低漏撿。

### 遺跡地圖上方的寶箱數量

- 使用者疑問：遺跡裡面開地圖，上面顯示的寶箱數量是什麼；是否代表這裡還有寶箱。
- 目前狀態：待實機確認，不可先講死。
- 後續要查：地圖顯示的是區域寶箱總數、已開啟數、剩餘數，還是目前可取得數量。
- 後續攻略用途：如果能確認，這會是很適合新手攻略的 `不要漏寶箱` 小技巧。

### NPC 對話獲得物品

- 使用者實機觀察：有時候跟路上 NPC 對話也會獲得物品。
- 後續要查：哪些 NPC 會給、是否一次性、是否和主線進度或時代有關、會不會錯過。
- 後續攻略用途：可以提醒新手不要只追任務標記，路上 NPC 有時候也值得順手對話。

### 買前補充：像薩爾達的生命碎片成長

- 使用者買前觀察：很像薩爾達，生命值要蒐集生命碎片提升上限。
- 已有媒體補強：GNN 試玩報導提到本作沒有一般經驗值升級，艾略特能力固定，生命力透過收集生命碎片合成生命之露提高上限。
- 買前稿用途：放在玩法定位或探索成長章，幫觀眾理解這不是刷等級型 RPG，而是探索收集型成長。
- 可用口播：`它的生命值成長很像 2D 薩爾達，不是靠打怪升級，而是靠探索蒐集生命碎片，把生命上限慢慢撐起來。`

### 其他 Square Enix 遊戲存檔連動：買前 / 新手攻略都可提醒

- 核心提醒：買正式版前要先確認自己要買在哪個平台，因為特典要求同一台主機、同一平台或同帳號的符合資格存檔 / 購買紀錄；雲端存檔可能需要下載到主機。
- 官方注意：部分平台可用購買紀錄判斷；有體驗版的對應遊戲，Demo 存檔也可能符合資格；開始遊戲後再補對應存檔，也可以領。
- 領取位置：標題畫面選 `Promotional Bonus` 連動；進遊戲後調查 `護佑時代 / Age of Safekeeping` 孤兒院二樓的桌子，從領取獎勵選單領。
- Octopath Traveler 0 / 八方旅人0：劍魔石 `Guidance of the Sacred Flame / 聖火的指引`，對生命值 30% 以下敵人，劍傷害 +10%。
- Dragon Quest I & II HD-2D Remake：劍魔石 `Critical Hit / 暴擊`，劍暴擊率 +6%。
- Bravely Default Flying Fairy HD Remaster：劍魔石 `Brave / 勇猛`，劍傷害 +10%，但防禦時盾牌耐久更容易下降。
- Various Daylife / 日常生活：劍魔石 `Expeditionary Wisdom / 探索禮儀`，劍蓄力時間 -10%。
- Triangle Strategy / 三角策略：劍魔石 `Counter Stance / 反擊準備`，盾牌耐久 90% 以上時，劍傷害 +10%。
- Trials of Mana / 聖劍傳說3 Trials of Mana：`Laurentian Plumage / 羅蘭的羽毛飾品`，持有後可在城鎮遇見某人；偏彩蛋 / 事件觸發，不是純戰力。
- 新手攻略用途：做正式上線後新手攻略時，可以提醒玩家一開始先到標題畫面領連動，再進孤兒院二樓桌子領獎勵。

### 雙倍掉落是什麼

- 狀態：使用者指定待研究，尚未查證完整機制。
- 後續要查：雙倍掉落是魔石效果、護符 / 配件效果、狀態效果、難度差異，還是特定敵人 / 區域規則。
- 後續要查：觸發條件、持續時間、掉落種類、是否影響魔石碎片、金錢、素材或道具。
- 後續要查：是否能和其他掉落加成疊加，前期是否值得優先取得。

### 裝備多種武器

- 狀態：已知本作有 7 種武器，官方 system 頁與 GNN 試玩報導都提到一次可裝備 2 種武器，並可切換。
- 使用者買前 / 新手實機補充：遊戲中只能八方向攻擊，長槍、弓箭這種直線武器打斜向敵人要瞄準，容易揮空；劍是扇形攻擊，目前體感最好命中、最適合前期穩定使用。
- 後續要查：新手前期最推薦帶哪兩種武器；目前先把 `劍` 當穩定上手武器，把長槍 / 弓定位為需要對位但刷怪或遠距有價值。
- 後續要查：哪些武器適合探索，哪些適合刷魔石 / 清怪 / 打王。
- 後續要查：武器和魔石效果如何搭配，哪些組合最適合初期玩家。

### 藥水

- 狀態：GNN 試玩報導提到消耗性道具可在商店購買，買藥水需要自購玻璃管當容器。
- 後續要查：玻璃管取得方式、價格、數量上限、是否重複使用。
- 後續要查：藥水補量、使用限制、和死亡復活 / 存檔點回血的取捨。
- 後續要查：新手開局應不應該先買玻璃管和藥水。
