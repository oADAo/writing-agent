## 可長期沿用的判斷

- 影片定位應該是 `買前必看前置流程`：先給玩家疑慮池與可挑章節，不直接寫完整稿。
- 核心切角：`不要只因為 HD-2D 就首發，先用 Prologue Demo 判斷操作、探索與夥伴提示能不能接受。`
- 遊戲體感應放在 `老派俯視角動作 RPG` 脈絡，不要讓觀眾誤以為是 Octopath 類回合制 RPG。
- 使用者修正：正式章節要以遊戲本身為主，多介紹遊戲內系統和好不好玩；Steam / PC / Heuria 風險要保留，但不要搶走主體。
- 使用者 Demo 實機補充：人物是像素，但劇情對話有完整可動立繪，而且都有語音，沉浸感十足；正式稿要把它當成畫面與劇情演出亮點講，不要只寫 HD-2D 很漂亮。
- 使用者 Demo 疑問：難度差異要補清楚。暫時整理為受傷量、敵血、敵人數量、接觸傷害、道具價格、復活費用、存檔點回血；繁中 UI 名稱與 Hard 存檔點回血規則待實機確認。
- 使用者鎖定必講：基本即時動作戰鬥一定要介紹，包含走位、攻擊方向、盾牌、防禦時機、武器切換、魔石 / 被動。
- 使用者買前武器補充：遊戲只能八方向攻擊，長槍、弓箭等直線武器打斜向敵人需要瞄準；使用者目前體感最好用的是劍，因為扇形攻擊比較容易命中。
- 使用者 Demo 實機補充：魔石系統可以讓武器變強、加效果；正式稿要用這句玩家能懂的語言解釋魔石，不要只講星等、抽取或刷素材。
- 使用者買前補充：體感很像薩爾達，生命值要蒐集生命碎片提升上限；可放在玩法定位或探索成長章。
- 使用者買前補充：有類似薩爾達神殿的 `生命神殿`，裡面可能是一個謎題或挑戰，解完後可拿生命碎片；可放在探索 / 成長章。
- 使用者買前補充：生命神殿實機體感很像薩爾達小神殿；目前看到陷阱操作型的 `鍛鍊之間`，以及放一堆怪物、要全清的 `力量試煉`。
- 買前必看新增提醒：指定 Square Enix RPG 的同平台 / 同帳號存檔或部分平台購買紀錄可領連動獎勵，包含八方旅人0、HD-2D 版勇者鬥惡龍 I&II、勇氣默示錄、日常生活、三角策略、聖劍傳說3 Trials of Mana；可放在 Demo / 買哪個平台章，提醒觀眾先確認自己存檔在哪個平台。
- 後續新手攻略待研究：使用者指定先記 `雙倍掉落是什麼`、`裝備多種武器`、`藥水`、`設定調搭檔對話頻率 / 關自動過場`、`大地圖與小地圖資訊`、`掉落物是否自動收進背包`、`遺跡地圖寶箱數量`、`NPC 對話獲得物品`。另存 `workspace/memory/games/elliot-millennium-tales/20260529-new-player-guide-notes.md`。
- 新手攻略新增確認：地圖會顯示迷宮剩餘寶箱數；Drop Bonus 是不受傷連續殺怪提升掉落，使用者觀察 5 殺 1 星、15 殺 2 星；部分支線有時效，主線推過可能不能接，任務清單可看目前支線。
- 新手攻略新增確認：`蓮藕護符` 可以讓掉落獎勵不會因為受到傷害而中斷一次；適合放在刷錢 / 刷魔石 / Drop Bonus 章節，取得方式與是否消耗待查。
- 新手攻略新增確認：地上掉落物不撿會消失，不會自動進背包；打完 Boss 掉落一堆時要趕快撿。連擊殺怪後角色身上會顯示目前幾 `CHAIN`。
- 新手攻略新增問答：護佑時代擋路的裂牆 / 特定石頭優先試炸彈，木樁類等鎚子，火把 / 冰等菲伊魔法，有些寶箱或通路體驗版目前拿不到；刷錢刷魔石核心是維持 Drop Bonus，路線可看迷霧遺跡、白色區域北部 / 北洞、沙漠洞窟。
- 新手攻略新增問答：`極光幕` 若是王國周邊 / 野外邊界的彩色光幕，先視為加護魔法結界或劇情阻擋，不能用武器 / 炸彈硬破；`地上大洞` 若是會吸人的洞，丟炸彈進去破壞，若是跳不過的坑則等夥伴傳送 / 高速移動或正式版能力。
- 使用者鎖定內容系列順序：1. `買之前必看`；2. `體驗版先做`；3. `正式上線後的新手攻略`；之後再依實際熱點追加。系列規劃另存 `workspace/memory/games/elliot-millennium-tales/20260529-content-series-plan.md`。
- 建議章節主軸：遊戲定位 -> 像素角色與動態立繪 -> 基本即時戰鬥 -> 武器系統 -> 魔石 build -> 探索 -> 四時代解謎 -> 難度與存檔 -> Demo 可玩內容 -> 好不好玩總評 -> 適合誰 / 誰該等。
