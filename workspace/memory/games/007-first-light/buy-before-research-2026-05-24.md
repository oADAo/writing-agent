## 這輪可重用結論

- 片名不要只寫死一個中文譯名。Steam 台灣頁目前用 `《007 曙光初現》`；H2 / 巴哈姆特 GNN 實體版新聞用 `《007 初露鋒芒》（007 First Light，舊暫譯：007 曙光初現）`；PS Store / Xbox 台灣頁主要仍顯示英文 `007 First Light`。安全寫法是：`007 First Light（Steam 譯：007 曙光初現；H2 / 巴哈譯：007 初露鋒芒）`。
- 核心定位不要講成 `Hitman 4`。比較安全的判斷是：IO Interactive 把 Hitman 的潛行、場景路線、道具互動與社交潛入 DNA，放進更快、更電影式、更任務推進導向的第三人稱 007 動作冒險。
- 最適合買前片的主軸是 `Hitman 期待管理 + 007 電影感 + 首發風險`。觀眾真正需要知道的是：自由度在哪裡、哪裡會收回線性演出、PC / Denuvo / Switch 2 / Day-one patch 有哪些買前風險。
- 中文圈和 B 站大量用 `殺手 47 / Hitman / 刺客任務 / 德瑞克 / Uncharted` 來理解這款；英文圈則常見 `Hitman meets Uncharted`、`not quite Hitman`、`more action than sandbox`；日文媒體多把它包成 `Hitman DNA + 007 らしいスパイアクション`。
- PC 版要特別保守：Steam 顯示 Denuvo Anti-Tamper；Reddit / Steam Community / PC 媒體都有 Denuvo、效能、離線保存性的疑慮。正式稿不能保證首發 PC 表現。
- Switch 2 版不可跟其他平台一起當作同日首發。官方 FAQ / Nintendo / H2 / GNN 都指向 Switch 2 是 2026 年夏季或未定，晚於 PS5 / Xbox / PC 的 2026-05-27。

## 已確認事實鎖定

- 開發 / 發行：IO Interactive A/S；官方 FAQ 說與 Amazon MGM Studios 合作。
- 發售：PS5 / Xbox Series X|S / PC 預定 2026-05-27；Nintendo Switch 2 預定 2026 年夏季或 TBD。
- 類型：官方與商店頁描述為單人第三人稱動作冒險 / 劇情導向 007 起源故事。
- 多人：官方 FAQ 確認沒有 multiplayer。
- 網路需求：官方 FAQ 說 day-one patch 需要一次網路連線；之後主線可離線玩；排行榜、挑戰與更多內容需要線上。
- Steam 台灣：`《007 曙光初現》`，NT$1,790，支援繁中介面與字幕，英文完整語音，Denuvo Anti-Tamper。
- PlayStation 台灣：PS5 版 NT$2,090，預購含免費 Deluxe Upgrade，24 小時搶先遊玩、服裝與金色道具包，標示繁中 / 簡中 / 英文 / 日文 / 韓文。
- Xbox 台灣：NT$1,790 起，Xbox Series X|S Enhanced、Xbox Play Anywhere、單人、Xbox achievements。
- Switch 2：Nintendo UK 顯示 release date TBD、1 player、Save Data Cloud；H2 日文頁提到 Switch 2 實體 key card 首次下載需網路與 75GB 以上空間。

## 玩法判斷

- 官方確認：潛行、徒手 / 武器戰鬥、Q 裝備、載具、社交潛入、任務重玩 / 挑戰或 TacSim 相關內容。
- 媒體試玩回報：冰島偏教學與電影式導入、馬爾他偏 MI6 訓練、Kensington / 博物館任務最接近 Hitman 式寬線性潛入。
- 媒體試玩回報：多數媒體認為它不是完整 Hitman 沙盒，而是有開放解法區塊的線性 / 寬線性動作冒險。
- 媒體試玩回報：License to Kill 不是自由殺人，而是敵人具致命意圖時才讓 Bond 使用致命武力，適合拿來解釋 007 跟 47 的核心差異。
- 需要上市驗證：實際任務數、TacSim 深度、重玩性、AI、槍戰手感、追車比例、正式版效能、Switch 2 表現、Denuvo 影響。

## 禁講或保守講法

- 不要說它就是 `007 版 Hitman` 或 `Hitman 4`。
- 不要保證有完整換裝偽裝沙盒、完整任務箱庭、完整自由殺法。
- 不要把媒體 20 小時說法當官方定案；只能說媒體轉述 / 試玩圈資訊。
- 不要保證 Switch 2 首發同步、掌機效能、PC 優化、Denuvo 無影響。
- 不要把官方宣傳語直接翻成玩家體驗；正式評價要等上市後 review / 實機。
