# Game Memory

## Game Overview

- `Pokémon Pokopia` 是 Nintendo Switch 2 的慢節奏生活模擬作品
- 官方確認全球於 `2026-03-05` 發售
- 目前已知這款遊戲的內容節奏不是只有開局，而是很快會往 `限時活動 / 收納 / 建築 / 搬家 / 夢島` 擴張

## Reusable Angles

- 首波或後續 `限時活動` 的 `別錯過 / 先換什麼 / 先抓誰`
- `收納 / 工作台 / 預製屋 / 主倉城` 的效率拆解
- `搬家 / 棲息地配置 / 半自動農場 / 舒適度`
- `夢島 / 傳說 / 稀有素材 / 重置`
- `玩過後值不值得買 / 誰適合玩`
- 依 `2026-03-20` 重跑後，長片 Top 1 題型升級成：`倉庫 / 工作台 / 房子條件 / 豪窄 / 舒適度` 一次講清楚的效率片

## Reusable Shorts Angles

- `免拆小屋 / 房子判定 / 多層建築` 很適合做成 `少走彎路` Shorts
- `Rare Pokémetal / 3D 列印素材` 是英文社群高頻痛點，可轉成中文快解題
- `限時活動前置條件` 適合做成 `你現在不先做這個就來不及`
- `某個裝扮 / 家具 / 寶可夢快速拿法` 很適合量產型短影音
- 依 `真 Shorts + 小創作者優先 + 可複製熱門` 重新驗證後，跨語圈最穩的 Top 1 Shorts 題型是 `建築 / 房子 / 空間優化`
- 如果只看中文圈單支爆量，則 `隱藏事件 / 如何取得 X` 會更強，但跨語圈可複製性略弱
- 依 `2026-03-13` 到 `2026-03-20` 最近 7 天重跑後，Top 1 仍然是 `建築 / 房子 / 空間外掛`
- 這一週要特別記得分開兩件事：
  - `跨語圈最穩`：`建築 / 房子 / 空間外掛`
  - `中文圈單支最猛`：`隱藏事件 / 稀有天氣觸發`
- 日文這週公開可索引的 Shorts 樣本偏薄，所以週榜判斷主要靠 `英文 YouTube Shorts + 中文 YouTube / 抖音 / bilibili + 玩家社群`
- 如果時間窗再縮到 `2026-03-18` 到 `2026-03-20` 這 3 天：
  - `跨語圈最新最穩` 還是 `房子 / 空間外掛`
  - `中文最新鮮的新話題` 會改成 `隱藏家具 / 家具細節 / 家具彩蛋`
- 也就是說之後要先分清楚使用者要的是：
  - `最大`：做 `房子 / 小屋 / 空間外掛`
  - `最新`：做 `隱藏家具 / 家具細節`
- 依 `2026-03-23` 新補資料，`雷射感應 / 水門 / 自動瀑布門 / 水簾洞入口` 是值得追的 `建築奇觀 + 機關感` 題型
- 這題的跨平台公開訊號沒有 `豪窄 / 空間外掛` 那麼大，但新鮮度更高，適合在使用者明確指定 `酷機關 / 建築彩蛋 / 紅石感` 時優先使用
- 中文標題翻法不要太技術，優先翻成：
  - `水簾洞自動門`
  - `瀑布感應門`
  - `紅外線感應瀑布門`

## Known Audience Hooks

- 怕漏拿限時寶可夢或家具
- 怕現在亂蓋，後面整城拆掉重蓋
- 想知道預製屋是不是其實比自蓋更強
- 想把有用寶可夢集中到家附近，減少跑圖
- 想知道這款到底是 `舒服神作` 還是 `很快膩`

## Content Gaps Worth Reusing

- 中文圈目前較缺 `活動 deadline + 拿滿路線`
- 中文圈也缺一支真正把 `收納 / 工作台 / 預製屋 / 搬家 / 舒適度` 串成一條效率邏輯的片
- 中文圈還缺一支真正把 `自建房判定 / 多層建築 / 免拆` 講清楚的短片
- 中文圈也還缺一支把 `建築奇觀` 翻成 `少走彎路 + 結果很離譜` 的 30 到 45 秒 Shorts
- 長片下一支最值得補的是：`Storage / 工作台 / 豪窄 / 房子條件` 的完整效率邏輯
