# Forza Horizon 6｜秘藏座駕 + 車房寶物記憶

更新日期：2026-05-22

## 可重用判斷

- 這題應區分兩套系統：`Treasure Cars / 秘藏座駕` 共 9 台，找到即可取得；`Barn Finds / 車房寶物` 共 15 台，需要 Discover Japan / 探索印章等級，找到後仍需修復等待。
- 台灣玩家用語 `車房寶物` 有巴哈支撐；Treasure Cars 對台灣觀眾應優先用 `珍藏車款`，`秘藏座駕` 是中文圈高搜尋詞但偏搜尋補詞。遊戲內台灣 UI 名稱仍需實機截圖確認。
- 巴哈截圖可見台灣 UI 底部有 `購買藏寶圖`、`收藏日誌`、`切換地圖區域`。
- 不能說「全部一開局就能拿」；只有 Treasure Cars 偏向探索到區域後可找，Barn Finds 仍受印章等級限制。
- IGN Barn Finds 頁面有風險：文字寫 14 台且漏掉 `Mitsubishi #1 Sierra Sierra Enterprises Lancer Evolution Time Attack`。完整車單應以 Red Bull / MapGenie / Eurogamer / Steam / Games.gg 交叉後的 15 台為準。

## 台灣繁中風險

- 區域譯名暫用：東京市、南野、大谷、島野山、北部、高城、外山、南岸、伊東；正式稿前需台灣實機 UI 核對。
- `Mitsubishi Montero Evolution` 不要直接改成 Pajero；可說「部分市場稱 Pajero Evolution」。
- `Mitsubishi #1 Sierra Sierra Enterprises Lancer Evolution Time Attack` 名稱來源差異大，正式稿前必須截圖核名。
- `Toyota 2000GT / 2000 GT`、`Nissan Skyline 2000GT-R / 2000 GT-R`、`Nissan R390 GT1 / R390 (GT1)`、`Lancer Evolution III / 3` 都需實機確認格式。

## 觀眾需求

- 中文圈高訊號：B站 9 台秘藏座駕影片約 40 萬觀看，台灣 / 繁中圈搜尋結果與攻略文常用 `珍藏車款`；留言抱怨某些攻略「遠景不清楚、地圖切太快、看完仍找不到」。
- 台灣圈已有巴哈 15 台車房寶物位置圖，但主要是截圖，缺逐台文字導航與入口教學。
- 影片差異點應是「台灣繁中可用 + 不買藏寶圖 + 每台都拍入口，不只貼點位」。
