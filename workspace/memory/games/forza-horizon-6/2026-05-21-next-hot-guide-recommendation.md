## Rationale

- 三語圈共同熱點是 `最強車 / 最快車 / Best cars how to get / AE86 FE / Mazda Furai / 各級別 meta`。
- 單純「刷抽獎」很熱，但與既有刷錢 / 刷技能點內容重疊；更適合成為最強車影片中的一段。

- Reddit / B 站留言顯示 AE86 FE、GT-R FE 類稀有車取得不穩，拍賣行 / 抽獎不宜當唯一推薦路線。

## Must Verify Before Script

- 繁中 UI 名詞：嘉年華遊戲清單、手環、秘藏座駕 / Treasure Cars、Furai、Forza Edition、Super Wheelspin。
