# Shorts Package

## Hook Title
不用拉水路，這款遊戲的菜田也能自己長

## Hook Burst Text
不用水路也能全自動

## Template Marked Script
```md
[hook] 你的農田，其實不用拉水路也能自己長。

[intro] 很多人一開始種田，第一個反應都是先鋪水路，然後自己拿水招一格一格澆。
[intro] 但我自己更推薦直接做這套半自動農田，因為蓄水桶裝滿水之後，會澆水的寶可夢真的會自己幫你顧附近的田。

[chapter] 第一步 先把田整理成十六格。
這套做法我會直接抓兩隻寶可夢來分工，一隻大概顧八格，所以十六格是最剛好的大小。
這樣你後面不管是種菜、補種子，還是想再往外擴，整塊田都比較穩。

[chapter] 第二步 旁邊先放好蓄水桶。
這個道具本來就是拿來讓寶可夢幫你澆水的，所以一定要貼著農地放。
然後記得先用水招把它裝滿，不然就算寶可夢有能力，它也不會真的開始幫你澆。

[chapter] 第三步 再把會澆水和會栽培的寶可夢搬進來。
只要能力有對上，它們就會自己去顧附近的田，該澆水的時候澆水，該種的時候也會接著做。
後面你基本上只要回來收成、補種子，真的累了再餵一點樹果，整塊田就會順很多。

[ending] 所以這套最舒服的地方，不是農田看起來比較整齊而已，是你從此不用自己一格一格慢慢澆，種田這件事才真的開始幫你省時間。
```
