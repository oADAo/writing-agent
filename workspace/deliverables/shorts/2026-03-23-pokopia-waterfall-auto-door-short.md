# Shorts Package

## Hook Title
你還不會做超瞎趴的自動瀑布門嗎？

## Hook Burst Text
這根本就是水簾洞

## Template Marked Script
```md
[hook] 你還不會做超瞎趴的自動瀑布門嗎？

[intro] 這種門最帥的地方，就是你一靠近，上面的「水門」就會自己打開。
[intro] 整片「瀑布」直接掉下來，超像一個藏在山裡的「水簾洞入口」。

[chapter] 第一步 先把「瀑布本體」做出來。
正式開蓋之前，你要先把能吸水吐水的「搬水能力」解鎖，然後找一個你想做入口的位置。
我自己會先做一個一到兩格寬的「門洞」，再在正上方挖出淺淺的「水槽」，把兩邊牆也先補起來，這樣水才不會亂噴。
接著把水放進去，看它掉下來有沒有剛好蓋住整個「入口」，先把這片「水簾」喬順，後面再接機關。

[chapter] 第二步 先把最上面的「水門開關」裝好。
這一步我會直接在「水槽」的出水口放一個會開合的「窗蓋」或「水門」，讓它平常先把水卡在上面。
你可以先手動開一次、關一次，確認關著的時候水會停住，打開的時候整片「瀑布」會往下掉。
這個開關一定要先測好，因為它就是整套「自動門」最核心的零件。

[chapter] 第三步 再把下面的「雷射感應器」一路接上去。
第一顆「感應器」我會放在你人一定會踩進來的位置，讓你一靠近它就先吃到「移動」。
再來中間記得一直穿插會開合的「門」或「窗蓋」，因為感應器不能直接串感應器，你一定要靠這些會動的零件，一段一段把訊號往上送。
最後把最上面的開關接上去，再自己走一次，能不能感應、會不會開門、瀑布有沒有真的落下來，這三個都對了，你的「自動瀑布門」就算完成了。

如果你還只會做普通大門，真的可以開始試這種會自己打開的「瀑布門」，帥度直接差超多。
```
