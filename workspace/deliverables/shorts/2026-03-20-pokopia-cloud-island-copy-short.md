# Shorts Package

## Hook Title
【很多人不知道】POKOPIA 官方島上的家具其實可以直接搬回家

## Hook Burst Text
家具可以搬回家

## Intro Overlay Text
- 很多人現在進官方雲島
- 真的都只是逛一圈而已
- 但這裡最值錢的其實不是參觀
- 是你喜歡的家具可以先拍再複製

## Step Cards
- `STEP 1` 先進官方雲島
- `STEP 2` 看到喜歡的先拍
- `STEP 3` 回家直接複製

## Teaching Subtitle Lines
- 先用能進雲島的方式進去。
- 看到喜歡的家具先不要急著走。
- 直接切到物件模式把它拍下來。
- 這些家具當下不能直接拿。
- 可是你回家之後可以再做出來。
- 因為 3D 列印可以把照片裡的家具複製出來。
- 所以等於官方直接開一整座樣品屋給你抄。
- 只是這招會吃素材。
- 我自己會先拍大型又難配的家具。
- 這樣一趟下來真的差很多。

## Final Short Script
官方島上的家具，其實可以搬回家。

很多人現在進官方雲島，真的就只是逛一圈而已，但這裡最值錢的根本不是參觀，是你看到喜歡的家具之後，可以先把它拍下來，回家再直接複製。

做法其實很簡單。先用能進雲島的方式進去，看到想要的家具先不要急著走，直接切到物件模式把它拍下來。這些家具當下不能直接拿，可是你回家之後，可以用 3D 列印把照片裡的家具做出來，所以等於官方把一整座樣品屋直接開給你抄。

只是這招會吃素材，所以我自己會優先挑那種真的難配、又很有存在感的大型家具先拍。這樣一趟下來，不只建築靈感有了，連家具都能順手帶一套回家，真的差很多。
