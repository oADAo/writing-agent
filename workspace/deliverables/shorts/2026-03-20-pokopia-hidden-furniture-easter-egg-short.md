# Shorts Package

## Hook Title
你以為 Pokopia 的家具只是裝飾？有些其實是彩蛋開關

## Hook Burst Text
家具其實是彩蛋開關

## Template Marked Script
```md
[hook] 你以為 Pokopia 的家具只是裝飾？有些其實是彩蛋開關。

[intro] 很多人現在看到「家具」，第一個反應都是加「舒適度」。
[intro] 但這遊戲更狠的地方，是你把某些「家具」擺對，會直接開出「隱藏互動」甚至「特殊棲息地」。

[chapter] 第一步 先別把家具只當擺設。
像「玩具」、「風車」、「積木」、「沙包」這些東西，真的不是放著好看而已，寶可夢會自己跑去玩、去轉，甚至直接做出一堆你平常看不到的「互動動作」。
所以有些房間看起來很普通，可是一開動之後，整個畫面會突然活起來。

[chapter] 第二步 更狠的是有些家具組合會直接開條件。
像「皮卡丘沙發」和「皮卡丘娃娃」這種家具，單看很像只是可愛擺飾，可是你把它們擺在一起，它就不是裝飾了，而是會直接變成「特殊棲息地」的開關。
這種細節你如果沒特別去試，真的很容易整輪都直接錯過。

[chapter] 第三步 真正離譜的是它開出來的不是小特效。
這種家具彩蛋有時候會直接跳「Photo Op」通知，有時候甚至會把特定「Pokémon」叫出來。
也就是說，你以為自己只是少一件家具，實際上你可能是把一整個「隱藏內容」都漏掉了。

[ending] 所以 Pokopia 最容易被忽略的彩蛋，不在地圖角落，而在那些你以為只是裝飾的「家具」裡。
```
