## 5 Shorts Topic Options
### Option 1
- `Topic Cluster`：免拆小屋 / 自建房判定 / 多層建築
- `Why It’s Moving`：
  - 中文短內容已經直接做成 `免拆小屋攻略`
  - 英文玩家連續在問怎樣才算房子、幾層才有效、最多住幾隻
- `Platform Evidence`：
  - 抖音精選 `宝可梦喵嗓`：`宝可梦pokopia超实用小技巧！免拆小屋攻略`
  - Reddit `Helpful tip!`：每層有門就可住 4 隻，環境評分也會拉升
  - Reddit `Building houses`：四面牆、一個門、三件家具才算房子
- `Cross-Language Evidence`：
  - 中文：`免拆小屋攻略`
  - 英文：`Building houses`、`Helpful tip!`
- `One-Line Topic Pitch`：很多人不是不會蓋房，是根本不知道遊戲怎樣才算你真的蓋好了。
- `Possible Hook`：你在 `Pokopia` 蓋半天都不算房子，通常不是你蓋錯，是你少了這 3 個條件。
- `Rhythm / Format Reference`：`前 1 秒直接說卡點` + `中段用 3 條規則連打` + `最後補一個多層偷空間技巧`
- `Chinese Audience Fit`：高，因為有立即收益，也能自然講成 `少走彎路`
- `Use / Skip`：`Use`

## Top 1 Recommendation
- 題目：`Pokopia 很多人蓋半天都不算房子，因為少了這 3 個條件`
- 推薦原因：
  - 最符合 `小帳號也能跑得動`
  - 同時有中文短內容與英文社群高頻痛點
  - 不靠新聞，不靠大媒體，不靠官方預告
  - 能在前 `1-3 秒` 直接成立 hook
  - 很適合轉成 `玫玫物語` 會講的 `少走彎路` 語氣
- 建議 hook：
  - `你在 Pokopia 蓋半天不算房子，通常不是你蓋醜，是你根本沒達成這 3 個條件。`

## Why Now
- 遊戲剛發售兩週左右，玩家還在第一輪 `開荒 / 蓋房 / 收納 / 活動`
- 小創作者已經開始把痛點切成超短攻略題
- 現在做 `少走彎路` 題比做大而全介紹更容易吃到點擊

## Risks / Unknowns
- `YouTube Shorts / IG Reels / 巴哈姆特 / 日文個人創作者` 這輪雖然有查，但可直接驗證的小創作者樣本還不夠厚
- 因此這次主結論主要建立在 `抖音精選 + bilibili + Reddit` 的小創作者與玩家社群交叉驗證
- 如果要再提高準確度，下一輪最好直接進平台內搜 `YouTube Shorts` 與 `IG Reels` 的原生短片
