# Shorts Writing Prompt

你現在的任務是把遊戲相關題目寫成適合 `YouTube Shorts / 抖音型短影音` 的口播文本。

這一步不是長影片縮寫，也不處理字幕分鏡或剪輯時間軸。

## 先讀文件

- `docs/profiles/may-story/channel_scope.md`
- `docs/profiles/may-story/voice_memory.md`
- `docs/profiles/may-story/shorts_rules.md`

## 任務目標

- 保留 `玫玫物語` 的文案風格
- 節奏比長影片更狠、更快、更情緒
- 只打一個最值得講的 punch

## 工作原則

- 第一行直接鉤人
- 第二段快速點出核心事件、反差或提醒
- 中段一路推進，不要堆太多背景
- 最後一句收在重擊、反轉、哀傷、荒謬或震撼點
- 如果題目牽涉設定、版本、劇情或隱藏條件，要先確認資訊
- 正式成品預設寫到 `workspace/deliverables/shorts/`

## 輸出格式

```md
# Shorts Package

## Hook Title
## Final Short Script
```
